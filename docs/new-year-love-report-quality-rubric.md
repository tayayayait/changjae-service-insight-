# 2026 연애/결혼운 결정도구형 품질 채점표

## 목적
- 대상: `saju-love-focus`
- 목표: 감정 해석 중심 리포트를 넘어, 사용자가 실제 관계 의사결정을 할 수 있는 수준으로 품질을 관리한다.
- 총점: `100점`

## 공통 채점 항목
| 항목 | 배점 | 판정 기준 |
|---|---:|---|
| 핵심 질문 일치도 | 15 | 관계의 속도/방향/결혼 전환 판단에 직접 답하는가 |
| 결정 도구 실용성 | 20 | 진행/보류/중단 기준이 실제 의사결정에 쓸 수 있는가 |
| 실행 가능성 | 20 | 조건 + 행동 + 확인 기준 문장이 충분한가 |
| 근거 연결성 | 15 | 사주 근거와 현실 행동 기준이 문장으로 연결되는가 |
| 분기 해석 완성도 | 15 | 각 분기에 기회/리스크/판단 기준/실행/실수 회피가 드러나는가 |
| FAQ 해소력 | 15 | 소비자 질문 8개에 구체 답변이 제공되는가 |

## 필수 통과 기준
- `marriageDecisionBoard` 3개 이상
- `meetingChannelPriority`, `greenFlagChecklist`, `redFlagChecklist`, `conflictProtocol` 각 2개 이상
- `consumerFaq` 8개 고정, 모든 항목에서 `question/answer` 누락 없음
- `analysisBlocks`에서 각 분기 `actionStrategy` 2개 이상

## 감점 규칙
- 추상 문장 감점 (-8~-15): "좋아진다/조심해라" 수준의 근거 없는 문장 반복.
- 결정 기준 누락 감점 (-6~-12): 행동 문장에 확인 기준이 없거나 측정 방식이 불명확함.
- 분기 중복 감점 (-4~-8): 분기별 문장이 사실상 동일해 시기별 판단 차이가 없음.
- FAQ 피상 감점 (-4~-8): 질문은 있으나 답변이 실행 기준 없이 일반론에 머묾.

## 가점 규칙
- 전환 판단 명확성 (+6~+12): 진행/보류/중단 시그널이 명확히 분리됨.
- 갈등 복구 실전성 (+4~+8): 24h/72h 프로토콜이 실제 대화 순서로 제시됨.
- 채널 전략 정밀도 (+3~+6): 만남 채널별 주간 실행과 체크 기준이 구체적임.

## 운영 기준
- 권장 최소 점수: `85점`
- 80점 미만: 프롬프트 재생성 또는 fallback 규칙 보강
