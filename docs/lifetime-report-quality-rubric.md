# 인생 총운 7종 리포트 채점표

## 목적
- 대상: `saju-lifetime-roadmap`, `saju-daeun-shift`, `saju-career-timing`, `saju-wealth-flow`, `saju-helper-network`, `saju-energy-balance`, `saju-yearly-action-calendar`
- 목표: 생성된 리포트가 "읽을 만한가" 수준이 아니라 "돈 받고 제공 가능한가"를 점수로 판단한다.
- 총점: `100점`

## 공통 채점 항목
| 항목 | 배점 | 판정 기준 |
|---|---:|---|
| 핵심 질문 일치도 | 10 | 서비스 고유 질문에 정확히 답하는가 |
| 해석 밀도 | 15 | 요약이 아니라 인과와 구조가 드러나는가 |
| 실행 가능성 | 15 | 오늘/이번 주/이번 달 수준으로 행동이 구체적인가 |
| 근거 명확성 | 10 | 왜 이런 해석이 나왔는지 문장으로 드러나는가 |
| 서비스 차별성 | 15 | 다른 6개 리포트와 문장/주제/처방이 겹치지 않는가 |
| 시간축 적합성 | 10 | 해당 서비스 시간 해상도에 맞게 설명하는가 |
| 가독성 | 10 | 섹션 흐름이 매끄럽고 문장이 과하게 모호하지 않은가 |
| 보강 섹션 품질 | 15 | 심화 인사이트, 심화 해석 포인트, 실행 프로토콜, 체크 질문, 시각 설명이 살아 있는가 |

## 서비스별 가산/감점 포인트

### 1. 인생 총운 로드맵
- 가산:
  - 직전→현재→다음 연결이 자연스럽다
  - 인생 단계 서사가 보인다
  - 성숙/확장/정리 축이 분리된다
- 감점:
  - 커리어/재무 세부 조언으로 흐른다
  - 장기 흐름이 아니라 일반 총평처럼 읽힌다

### 2. 대운 전환 시그널
- 가산:
  - 전환 전/중/후 차이가 분명하다
  - 오판 방지 포인트가 실질적이다
  - 90일 안정화 플랜이 구체적이다
- 감점:
  - 그냥 "변화가 온다" 수준에 머문다
  - 장기 로드맵 설명을 반복한다

### 3. 커리어 타이밍 리포트
- 가산:
  - `결정할 것/보류할 것`이 명확하다
  - 역할 적합성과 환경 적합성이 분리된다
  - 기회비용이 설명된다
- 감점:
  - 재물/관계 일반론으로 새어 나간다
  - 실행 시점이 불명확하다

### 4. 재물 흐름 레이더
- 가산:
  - 인생 자산 사이클 4단계(축적/확장/방어/변동)가 명확히 보인다
  - 유입/누수 구조와 단계별 운영 규칙이 연결된다
  - 금지 판단과 손실 한도, 전환 신호가 함께 제시된다
- 감점:
  - 단순히 "돈이 들어온다/조심해라" 수준이다
  - 특정 연도(예: 2026) 반복으로 장기 서사가 약하다
  - 커리어 조언과 중복된다

### 5. 관계·귀인 네트워크
- 가산:
  - 가까운 관계/협업/사회 레이어가 나뉜다
  - 귀인 유형이 실제 인물상처럼 읽힌다
  - 갈등 루프 차단 문장이 쓸모 있다
- 감점:
  - 관계 일반론만 반복한다
  - 에너지/재무 조언과 경계가 흐리다

### 6. 적성·에너지 밸런스
- 가산:
  - 몰입 트리거와 소진 신호가 동시에 보인다
  - 24h/72h 회복 프로토콜이 실용적이다
  - 하루/주간 리듬 설계가 구체적이다
- 감점:
  - 단순 성격 설명에 머문다
  - 연간 계획표처럼 서술된다

### 7. 연간 실행 캘린더
- 가산:
  - 분기 목표와 월별 행동이 연결된다
  - 푸시/주의 구간이 분명하다
  - 밀림 복구 플랜이 실제 일정 관리에 쓸 만하다
- 감점:
  - 총운 요약 반복에 머문다
  - 실행 체크포인트가 추상적이다

## v2.8 Yearly Calendar Roadmap Rubric (2026-03-28)
- target service: `saju-yearly-action-calendar`

### Deductions
- Single-year schedule bias (-8 ~ -15): only quarter/month checklist is present and lifecycle transition context is missing.
- Missing long-horizon fields (-6 ~ -12): any of `yearToLifeBridge`, `lifecycleExecutionPattern`, `phaseFocusMap`, `accumulationTransitionFlow`, `longPracticeStrategy` is missing.
- Quarter-month mismatch (-4 ~ -8): quarter-month mapping is inconsistent (Q1=1~3, Q2=4~6, Q3=7~9, Q4=10~12).
- Mechanical tail repetition (-3 ~ -6): completed action sentences repeatedly get appended with boilerplate tails.
- Internal token leakage (-3 ~ -6): user-facing output exposes internal tokens such as `calendar-map`, `Q1~Q4`, `Structuring`, `Acceleration`.
- Missing yearly quality gate (-4 ~ -8): validator+fallback pass is not applied in yearly parse stage and normalize final stage.

### Additions
- Lifecycle-year bridge quality (+6 ~ +12): sentence structure keeps `current position -> long-term objective -> current-window action -> checkpoint`.
- Phase focus clarity (+4 ~ +8): 0~2y / 3~5y / 6~10y / transition are expressed as distinct execution patterns.
- Accumulation-transition discipline (+3 ~ +6): `build/drop/transition trigger/recovery rule` are separated with concrete decision rules.
- Cache invalidation discipline (+3 ~ +6): yearly prompt/template version and cache version are raised together (`reportTemplateVersion` + `cacheVersion` + `yearlyQualityGateVersion`).

## v2.9 Yearly Total-Fortune Rubric (2026-03-28)
- target service: `saju-yearly-action-calendar`

### Hard Gates
- 9-section structure must render in fixed order:
  - `한 줄 총평`
  - `지금 인생의 큰 흐름`
  - `올해의 의미`
  - `올해 이후 10년의 흐름`
  - `장기 패턴 해석`
  - `올해의 핵심 테마 3가지`
  - `분기별 실행 캘린더`
  - `올해가 끝났을 때 남아야 할 것`
  - `한 줄 결론`
- `keyThemes` must be exactly 3 items.
- `quarterNarratives` must be exactly 4 items with distinct quarterly role/meaning/focus/caution.
- Narrative layer (`oneLineTotalReview/currentLifeFlow/meaningOfThisYear/tenYearFlow/longPatternInterpretation/keyThemes/yearEndResidue/closingLine`) must not include short-term markers.
- Yearly supplement cards `오늘/이번 주 실행`, `이번 달/주의 항목` must not render.

### Deductions
- Narrative collapse to checklist tone (-8 ~ -15): long-horizon fields are written as immediate execution orders.
- Short-term marker leakage (-6 ~ -12): narrative layer includes `오늘/이번 주/이번 달/3월 말까지/매일 30분`.
- Quarterly role duplication (-4 ~ -8): quarter narratives repeat the same role/meaning across multiple quarters.
- Legacy short-term supplement exposure (-4 ~ -8): yearly UI still shows common executionProtocol cards.

### Additions
- Long-horizon coherence (+6 ~ +12): yearly meaning and 10-year flow are causally connected.
- Quarter identity clarity (+4 ~ +8): each quarter clearly differentiates role/meaning/focus/caution.
- Year-end residue quality (+3 ~ +6): output specifies what must remain after the year beyond short-term tasks.
