# Astrology 정보 수정 단일 화면 전환 (2026-03-20)

## 목적
- `/astrology`의 `정보 수정` 진입 시, 출생 정보와 출생지 선택이 분리된 2-step 구조를 1개 화면으로 통합한다.
- 입력 흐름을 단축하고, 사용자가 한 번에 전체 값을 확인/수정할 수 있게 가독성을 개선한다.

## 변경 내용
- `src/pages/AstrologyPage.tsx`
  - `출생 정보 수정(step 1)`과 `출생지 선택(step 2)`을 하나의 편집 카드로 통합.
  - 편집 카드 내 입력 순서를 `이름 → 생년월일 → 출생시간 → 출생지`로 고정.
  - 생년월일/시간 입력에 라벨 및 범위(`month 1~12`, `day 1~31`, `hour 0~23`, `minute 0~59`)를 명시.
  - 하단 액션을 `이전` + `리포트 생성` 2버튼으로 단순화.

## 기대 효과
- 정보 수정 과정에서 화면 전환이 1회 줄어든다.
- 입력 값 누락 여부를 같은 화면에서 즉시 확인 가능하다.
- 출생지 선택이 동일 카드에 포함되어 작성 맥락이 끊기지 않는다.
